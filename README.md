# Introduction

This project is a website dedicated to the character Sakura Haruno. The idea is to create a simple and attractive space where fans can learn more about her story, abilities, and evolution throughout the anime.

# Target Audience

This website is mainly aimed at:

- Anime fans, especially Naruto fans  
- People interested in character analysis  
- Teenagers and young users who enjoy anime content  
- Anyone who wants to learn more about Sakura Haruno in a quick and simple way  

# Inspirations and Creative Process

For this project, I took inspiration from:

- Anime fan pages that provide detailed character information  
- Websites with clean and organized layouts  
- Platforms that combine text and images in a visually appealing way  

Creative process:

First, I thought about what kind of content a Naruto fan would like to see. Then, I organized the information into simple sections like biography, abilities, and trivia. Finally, I focused on making the design clear and easy to navigate, avoiding too much clutter.

# References and Additional Resources

Some useful references for this project include:

- Official Naruto content and character descriptions  
- Anime-related websites and fan pages  
- Online images and media related to Sakura Haruno  

# Technologies

- HTML for structure  
- CSS for styling  
- JavaScript for basic interactivity  

# Objective

The main goal is to create a simple, interesting, and user-friendly website that makes it easy to explore and enjoy content about Sakura Haruno.
